---

# 📖 Project Overview

This project analyses pizza sales data to provide meaningful business insights into revenue, customer ordering behaviour, product performance, and sales trends.

The dashboard transforms raw transactional data into interactive reports that help stakeholders monitor KPIs, identify high-performing products, and support data-driven business decisions.

# ✨ Features

The dashboard enables users to:

* Monitor key sales KPIs
* Analyse daily and monthly sales trends
* Explore pizza category performance
* Analyse pizza size preferences
* Identify best-selling products
* Identify underperforming products
* Filter reports by pizza category
* Filter reports by date range
* Navigate between dashboard pages
* Compare sales performance interactively

---

# 📊 Dashboard Pages

### 📌 Sales Overview

Provides a high-level overview including:

* Total Revenue
* Average Order Value
* Total Orders
* Total Pizzas Sold
* Average Pizzas per Order
* Daily Order Trends
* Monthly Sales Trends
* Sales by Category
* Sales by Pizza Size
* Category Performance Summary

---

### 🍕 Best & Worst Sellers

Provides detailed product analysis, including:

* Top 5 Pizzas by Revenue
* Bottom 5 Pizzas by Revenue
* Top 5 by Quantity Sold
* Bottom 5 by Quantity Sold
* Top 5 by Total Orders
* Bottom 5 by Total Orders
